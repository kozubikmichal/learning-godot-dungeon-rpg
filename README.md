# Dungeon RPG

A 2.5D dungeon RPG built with Godot Engine, following the GameDev.tv tutorial series.

## About

This project is a learning exercise created while following along with GameDev.tv's Godot game development course. The game features dungeon exploration, combat mechanics, and RPG elements in a 2.5D perspective.

## Features

- **2.5D Perspective**: Classic dungeon crawler view with modern mechanics
- **Combat System**: Engage in battles with enemies
- **Character Progression**: Level up and improve your character
- **Dungeon Exploration**: Navigate through procedurally designed dungeons
- **Inventory Management**: Collect items and manage equipment

## Getting Started

1. Make sure you have [Godot Engine 4.4](https://godotengine.org/) installed
2. Clone this repository
3. Open the project in Godot by importing the `project.godot` file
4. Press F5 or click the play button to run the game

## Controls

- **Movement**: WASD or Arrow Keys
- **Action**: Interact with objects and engage in combat
